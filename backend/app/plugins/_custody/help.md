# Custody Tracker

Track visitation events, expenses, and schedules with your child.

## Logging events

- "Picked up Mason" — logs a pickup stamped to now
- "Dropped Mason off at mom's at 7pm" — logs a dropoff with location
- "Mason called, no answer" — logs a missed phone call
- "Took a screenshot of a text from her" — opens text capture flow

## Logging expenses

- "Bowling with Mason, $42" — logs an activity expense
- "Bought Mason new shoes, $65" — logs a clothing expense
- "Doctor visit, $25 copay" — logs a medical expense

## Missed visits

- "She didn't bring him Saturday" — records a missed/denied visit

## Querying history

- "How much have I spent on Mason this month?" — expense summary
- "When did I last pick up Mason?" — last pickup
- "Show me all missed visits in January" — filtered event list

## Schedules

Set up a recurring custody schedule so JAIN can automatically flag missed visits. Use the Schedules screen in the Custody home to create an every-other-weekend or weekly schedule.

## Export

Export to CSV or PDF for court documentation. Use "Export my custody records" or open the Custody home and tap Export.
